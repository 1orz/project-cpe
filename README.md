# Project CPE - 5G CPE 高级后台项目

## 项目简介

这是一个为市面上成品 5G CPE（Customer Premises Equipment）设备开发的高级后台管理系统项目。本项目旨在为现有的 5G CPE 设备提供更多高级功能和可玩性，让用户能够更好地控制和定制他们的 5G 网络设备。
